# Yazi 단축키 가이드

## 네비게이션

### 기본 이동
- `j` 또는 `↓`: 아래로 이동
- `k` 또는 `↑`: 위로 이동
- `h` 또는 `←`: 상위 디렉토리로 이동 (뒤로 가기)
- `l` 또는 `→` 또는 `Enter`: 하위 디렉토리 진입 / 파일 열기

### 빠른 이동
- `g`: 첫 항목으로 이동
- `G`: 마지막 항목으로 이동
- `~`: 홈 디렉토리로 이동

## 파일 작업

### 선택
- `Space`: 파일 선택/선택 해제
- `v`: 비주얼 모드 (다중 선택)
- `V`: 비주얼 모드 (선택 해제)

### 복사/이동/삭제
- `y`: 복사 (yank)
- `x`: 잘라내기 (cut)
- `p`: 붙여넣기 (paste)
- `d`: **삭제 (휴지통으로 이동)** - 안전한 방법, 권장
- `D`: **영구 삭제** - 복구 불가능, 주의 필요
  - 확인 프롬프트에서 `y` (yes) 또는 `n` (no)로 응답

### 생성/이름 변경
- `a`: 새 파일 생성 (create file)
- `A`: 새 디렉토리 생성 (create directory)
- `r`: 이름 변경 (rename)

## 보기 옵션

### 표시 설정
- `.`: 숨김 파일 표시/숨기기 토글
- `s`: 정렬 방식 변경
- `z`: 현재 디렉토리 크기 계산
- `Z`: 선택한 항목들의 크기 계산

### 검색 및 필터
- `/`: 검색 (파일명)
- `f`: 파일 필터 (find)
- `Esc` 또는 `Ctrl + [`: 검색/필터 취소, 선택 해제

## 탭 관리

- `t`: 새 탭 생성
- `Tab`: 다음 탭으로 이동
- `Shift + Tab`: 이전 탭으로 이동
- `1`-`9`: 특정 탭으로 이동
- `Ctrl + c`: 현재 탭 닫기 (마지막 탭이면 yazi 종료)

## 작업 관리

- `w`: 작업(task) 목록 표시
- `Esc`, `Ctrl + [`, `Ctrl + c`, `w`: 작업 목록 닫기
- `X`: 복사/잘라내기 작업 취소

## 종료

- `q`: yazi 종료 (현재 디렉토리 저장)
- `Q`: yazi 종료 (디렉토리 저장 안 함)

## 도움말

- `?`: 도움말 보기

## 추가 기능

### 선택 반전
- `Ctrl + a`: 모든 항목 선택
- `Ctrl + r`: 선택 반전

### 셸 명령
- `:`: 명령 모드 진입
- `!`: 셸 명령 실행

## 주요 사용 패턴

### 파일/디렉토리 삭제하기
1. `j`/`k`로 삭제할 파일/디렉토리 선택
2. 여러 개를 삭제하려면 `Space`로 선택
3. `d` (휴지통으로 이동) 또는 `D` (영구 삭제)
4. 확인 프롬프트에서 `y` 입력
5. **권장**: 일반적으로 `d` 사용 (안전)

### 파일 프리뷰 후 뒤로 가기
1. 파일 선택 (프리뷰가 오른쪽에 표시됨)
2. `h` 또는 `←`를 눌러 상위 디렉토리로 이동
3. 또는 `j`/`k`로 다른 파일 선택

### 외부 프로그램으로 파일 열기
1. 파일 선택 후 `Enter` 또는 `l`
2. 외부 프로그램에서 파일 확인
3. 외부 프로그램 종료 (보통 `q` 또는 `Ctrl + c`)
4. yazi로 자동 복귀

### 다중 파일 복사/이동
1. `Space`로 파일들 선택
2. `y` (복사) 또는 `x` (잘라내기)
3. 대상 디렉토리로 이동
4. `p` (붙여넣기)
5. `X`로 작업 취소 가능

## 커스터마이징

단축키를 변경하고 싶다면 `~/.config/yazi/keymap.toml` 파일을 수정하세요.

기본 keymap 파일은 [공식 문서](https://yazi-rs.github.io/docs/configuration/keymap/)에서 확인할 수 있습니다.
